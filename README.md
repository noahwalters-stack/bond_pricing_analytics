**Bond Pricing Analytics Tool by Noah Walters**

This is a Python-based bond pricing calculator that computes metrics like YTM, accrued interest, durations, and tax-equivalent yield. The results are generated within an Excel report.

**Requirements**
- Python 3
- pip

**For Mac install:**
in terminal,
python3 -m pip install -r requirements.txt

For Mac run, open a terminal within the folder and input:
python3 bond_pricing_analytics.py --input sample_bonds.csv --output bond_report.xlsx



**For Windows install:**
in terminal/command prompt:
py -m pip install -r requirements.txt

For Windows run, open a terminal/command prompt within the folder and input:
py bond_pricing_analytics.py --input sample_bonds.csv --output bond_report.xlsx
